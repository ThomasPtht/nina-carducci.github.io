# NINA CARDUCCI - Débug et optimisation d'un site de photographe

![image](https://github.com/ThomasPtht/nina-carducci.github.io/assets/128037230/3703aab2-3215-4708-8151-e8055334b16a)

Demande de la cliente d'optimisation de son site selon ses demandes et selon le rapport détaillé qui a été réalisé pendant l'intervention (via Lighthouse). 
A savoir : 

- Optimisation des images du site : images redimensionnées et compressées, images renommées de manière explicite
- Ressources minimifiées et compressées (CSS, Bootstrap, JS), suppression du code inutile
- Accessibilité : attributs alt ajoutés, balises méta title/description/robots, modification de la structure des titres qui n'était pas optimale, ajout de balise lang, correction 
formulaire attribut for manquant. Modification couleurs background bouton afin d'améliorer le contraste, balises méta réseaux sociaux, balises référencement local ...
- Débogage du code (à l'aide de Devtools) : problème de navigation entre les images, background CSS non appliqué sur les filtres actifs.
- Rédaction d'un cahier de recettes et du rapport d'intervention
